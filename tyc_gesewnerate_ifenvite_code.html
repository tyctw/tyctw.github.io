使用說明

1. 此頁面顯示桃聯區會考落點分析邀請碼，每小時自動更新。

2. 邀請碼由後端 Google App Script 提供。

3. 點擊「複製」按鈕將邀請碼複製到剪貼簿。

4. 點擊「顯示QR碼」生成邀請碼的QR碼。

5. 點擊「分享邀請碼」啟動原生分享功能（若瀏覽器支援）。

6. 點擊「手動更新」可立即從後端獲取最新邀請碼。

7. 進度條顯示距離邀請碼更新的倒數進度。

8. 使用深色模式切換按鈕切換頁面主題。
